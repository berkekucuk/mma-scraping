# MMA Scraping Project

This is a hobby project developed to scrape MMA (Mixed Martial Arts) data for analysis and learning purposes.

## ⚠️ Ethical Disclaimer

**This project is purely a hobby project and has no commercial purpose.**

It is designed to be completely ethical and respectful of the data source provider's resources. The scraper is configured to place **minimal load** on the servers, adhering to polite scraping practices (e.g., respecting `robots.txt` where applicable, using appropriate delays, and limiting concurrent requests).

The goal is to gather data for personal educational use without disrupting the service or causing any harm.

## Tech Stack

*   **Python**: Core programming language.
*   **Scrapy**: Framework used for extracting the data.
*   **Docker + AWS Lambda**: Containerized serverless deployment for scheduled scraping jobs.
